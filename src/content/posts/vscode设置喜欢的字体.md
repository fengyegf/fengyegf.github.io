# 首先

你得有一个字体，打开并安装
![alt text](../img/vscode设置喜欢的字体/3.png)
然后找到windows目录下的字体目录
```txt
C:\Windows\Fonts
```
找到你安装的![alt text](../img/vscode设置喜欢的字体/4.png)
在打开vscode中的![alt text](../img/vscode设置喜欢的字体/1.png)
看到![alt text](../img/vscode设置喜欢的字体/2.png)
我们在前面添加
```txt
'字体名称',Consolas, 'Courier New', monospace
```
![alt text](../img/vscode设置喜欢的字体/5.png)

至于名称看着输入即可
