# 前言
我们本期使用的是来自[SAKURA FRP](https://www.natfrp.com/)

# 开始
首先注册一个账号进入后选择

- 服务->软件下载->选择对应的即可

# 配置

首先创建隧道，选择服务->创建隧道

这里要区分游戏和建站

- 如果你是游戏玩家想要和好友实现远程联机那么选择对应的udp或者tcp即可
- 如果你是用于穿透网站就选择可建站

按照对应的内容填写即可，建站需要备案，建议先给域名备案

# 最后开启即可
