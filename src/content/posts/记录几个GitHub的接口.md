# 发布正式版本
``` url
https://api.github.com/repos/仓库作者/作者项目名称/releases/latest
```
可以查询该项目的releases正式版本
# 全部版本
``` url
https://api.github.com/repos/仓库作者/作者项目名称/releases
```
可以查询发布的全部包
